# Pure TS!

Este projeto tem o objetivo de validar o uso de typescript sem bibliotecas de ui e o minimo de dependências na geração de interfaces estáticas.

- webpack
- jest
- SaaS

# Instalação

-- npm install
-- yarn install

# Dev

-- npm/yarn run start

# Test (tdd)

-- npm/yarn run test
-- npm/yarn run test:w //watch mode

# Deploy

-- npm/yarn run build
-- npm/yarn run build:dev
